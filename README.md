# px4_msgs

This package is derived from [my PX4 fork](https://github.com/kuederleR/PX4-Autopilot/tree/modalai_sitl). This version of PX4 messages is compatible with the PX4 version preinstalled on ModalAI Starling/Starling 2/Starling 2 Max. 

## Usage

1. Clone this repository into your workspace's `src/` directory.
2. From your workspace's root, run

``` bash
colcon build
```

3. Source the install script

``` bash
source install/setup.bash
```
